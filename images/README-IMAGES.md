# Images Needed for Fixla Website

Place these images in the `/images/` folder:

## Required Images:

1. **dog-walking.jpg** - Image of a dog being walked (for hero section)
2. **quick-services.jpg** - Image showing quick service work (person using smartphone)
3. **flexible-solutions.jpg** - Image of garden/yard work
4. **gutter-cleaning.jpg** - Image of gutter cleaning
5. **yard-raking.jpg** - Image of yard raking with leaves
6. **snow-removal.jpg** - Image of snow shoveling
7. **tire-change.jpg** - Image of tire/wheel change
8. **about-laptop.jpg** - Image of laptop on desk for about page

## Image Guidelines:
- Recommended size: 1200x800px or similar aspect ratio
- Format: JPG or PNG
- Optimize for web (compress to reduce file size)
- Use high-quality, professional images

## Where to get images:
- Use your own photos
- Free stock photo sites: Unsplash, Pexels, Pixabay
- Or use AI image generation tools

## Placeholder Colors:
Until you add real images, you can use colored boxes as placeholders:
- Green (#52c41a) for service images
- Gray (#888) for generic images
